# Overview
You are tasked with modeling the following basic lasercut joints using Fusion:
- Basic finger joint
- Kerf-adjusted finger joint
- Mortise and tenon joint with 2 tenons
- Snap joint
- Fastener joint
![Photo of the 5 joints: Basic finger joint, Kerf-adjusted finger joint, Mortise and tenon, Snap joint, and Fastener joint]()

# Learning Objectives
By the end of this activity you will be able to:
- Design several lasercut joints.

# Constraints
- Your joints must be modeled in place, with no move commands in your timeline. (-.1 points per move command)
- The joints should be in a single Fusion file, with each joint as a separate Component. (-.25 points)
    - You will need to use a Hybrid Design file to accomplish this.
- Use a material thickness of .125 inches.
- Each joint should be 2 bodies, not 1 (-.1 point).

# Deliverables
- Single accessible Fusion link
    - Make sure to click "Share the lastest change with anyone using this link:" and "Allow items to be downloaded." Leave "Require a password to access the link" unchecked.
    - All 5 joints 3D modeled in this file (must be Bodies, so you will need to extrude).

# FAQs
- **When I Extrude the second Body, it joins to the first, how do I prevent them from joining?**
- There are two ways to fix this:
    - Hide the first Body in the Browser before Extruding.
    - In the Extrude dialog box, select New Body instead of Join.
        - Sometimes an option will show up that allows you to pick which objects to Join to, but sometimes this isn't an option.
