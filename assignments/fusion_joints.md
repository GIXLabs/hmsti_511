# Overview
Fusion, and many other CAD programs, have options to prototype motion like hinges. This sort of modeling can help you assess whether your mechanism will work faster than physically manufacturing it. This short activity is designed to give you some experience with this functionality.

# Learning Objectives
By the end of this assignment, you will be able to do the following:
- Join components in Fusion to prototype motion.
- Select the appropriate 

# Constraints
- All of your joints must be in the same Fusion file.
- Each joint should be in its own Component, which then has the two Components that make up the joint.
- Each joint should have the appropriate dimensions described in the engineering drawings.

# Deliverables
- Using each of the following engineering drawings, model the following 3 joints and use the Joint command to mimic the mostion of the design.
    - Fixed joint combining two halves of an enclosure.
    - Slot joint for a lid.
    - Revolute joint for a hinge.
- Submit a shareable and downloadable Fusion link on Canvas.

# FAQs
- **Do I have to model the pieces of the joint in place, or can I use the Joint command to move them into the appropriate location?**
- Both options are ok, but you will likely find it easier to model them in place. Modeling in place allows you to use the other part as a reference.
