# Overview
To design objects that fit with other objects, you first have to be able to measure. Calipers are one of the most common measurement tools for products that are on the scale of consumer electronics. This activity will show you how to use calipers effectively.

# Learning Objectives
By the end of this activity you will be able to:
- Measure the exterior dimensions of an object
- Measure the interior dimensions of an object
- Measure depths using calipers

1. Calipers are commonly used tools for measuring precise lengths.
